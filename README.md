# UiPath Uncharted Website

This repository contains a single-page website for the UiPath Uncharted guest pitch.

## Files
- `index.html` — homepage content
- `styles.css` — modern visual styling
- `script.js` — mobile navigation toggle

## Deploying to GitHub Pages
1. Push this repository to GitHub.
2. In repository Settings > Pages, select the `main` branch and the `/ (root)` folder.
3. Save and visit the published GitHub Pages URL.
